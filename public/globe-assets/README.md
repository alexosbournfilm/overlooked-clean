Active globe topology source:
https://cdn.jsdelivr.net/npm/world-atlas@2/countries-10m.json

Derived from Natural Earth data via world-atlas. Used to render a high-detail flat Overlooked-gold country mask with transparent oceans.

`countries-mobile.json` is a downsampled derivative of `countries-10m.json` for the native mobile WebView globe. It keeps the same topology structure with fewer arc points so the globe can initialize and rotate smoothly on-device.

Legacy earth texture source:
